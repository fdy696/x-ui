# 介绍

### 项目演示地址

https://fdy696.github.io/x-ui/components/icon.html

### 开发目的

沉淀工作中遇到的需要自己封装的组件库，提高自己造轮子的技术，方便代码复用。

### 基本信息

X-UI 是一个基于 Vue 的 UI 组件库，目前包括有 ICon、Button、Timeline、Collapse、 Tree、 Calendar 组件；目前只对 icon 和 button 组件进行了单元测试，后续测试持续更新中...
