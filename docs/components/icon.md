# Icon图标

提供了一套常用的图标集合。

### 基本用法

<ClientOnly>
  <icon-demo/>
</ClientOnly>


### 示例代码

```vue
<x-icon icon="download"></x-icon>
<x-icon icon="awesome"></x-icon>
<x-icon icon="loading"></x-icon>
<x-icon icon="down"></x-icon>
<x-icon icon="up"></x-icon>
```

# Attributes


|参数| 说明 |  类型  | 可选值 | 默认值 |
| :-------------: |:-------------:| :-----:|:-----:|:-----:|
|icon| icon名称 | String |name| --
